Income Statement Analysis



• Context

This end-to-end Sales Analysis project was developed to address a real operational bottleneck within the company. The sales team used to perform manual data extraction and consolidation in Excel, a process that required an entire workday (around 8 hours) for each update. Moreover, the BI dashboard was refreshed only once per month, limiting visibility into ongoing sales performance.

The goal was to automate the entire process — from data ingestion to analytics — enabling near real-time updates and freeing analysts from repetitive manual tasks.

• Insights

A Power BI dashboard was connected directly to the warehouse, allowing continuous updates and interactive exploration of sales metrics such as revenue trends, product performance, and customer segmentation.

This project not only automated the data flow but also transformed the company’s decision-making process, making insights available in near real time.

• Technical Stack

  • Tools: FABRIC: DFGen2 | DW (SQL Server) | PowerBi
  • Data Source: ERP: PostgreSQL
  • Data Modeling:Medallion architecture and Star Schema with Fact and Dimension Tables
